Lab 5: Functions and Graphics

-----------
DESCRIPTION

In this lab, the program will perform some primitive graphics operations on a small simulated display. These functions will allow users to change the background color of the display, and “draw” horizontal and vertical lines on the display.
-----------
FILES

-
Lab5.asm

This file includes the assembly code of the lab.

-

This file include the test.

-----------
INSTRUCTIONS

This program is intended to be run using the MARS. To simulate a display, use the memory-mapped bitmap graphics display tool included with MARS.
