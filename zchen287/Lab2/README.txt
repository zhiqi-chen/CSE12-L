Lab 4: Simple Data Path

-----------
DESCRIPTION

In this lab, the user enters some numbers (0-F) using keypad. The program will print out the specific values on eight 7 segment displays (Register 0, Register 1, Register 2, Register 3, Keypad output, ALU Input 1, ALU Input 2, and ALU Output).

-----------
FILES

-
Lab2.lgi

This file includes the circuits we designed that has the same function as the instruction requires.

-----------
INSTRUCTIONS

This program is intended to be run using the Multimedia Logic (MMLogic or MML). While running, change the inputs on keyboard to see different outputs on 7 segment display.
